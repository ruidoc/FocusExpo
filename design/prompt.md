# FocusOne 设计稿生成提示词

> 基于 React Native 0.79.2 + Expo + iOS 16+ 的专注应用设计稿生成指南

## 🎯 设计目标

- **最低摩擦**：一键开始、路径最短、信息聚焦
- **状态清晰**：进行中/暂停/结束/未配置/需授权状态明显可感
- **可恢复**：冷启动可恢复当前专注态；文案一致、时间准确
- **一致性**：色板、间距、圆角、字号成体系；明暗主题一致

## 🎨 设计系统

### 色彩规范
```css
/* 主色系 */
--primary: #007AFF (iOS 蓝)
--primary-gradient: linear-gradient(135deg, #007AFF 0%, #5AC8FA 100%)
--success: #34C759 (专注模式绿)
--warning: #FF9500 (暂停橙)
--error: #FF3B30 (屏蔽模式红)

/* 中性色 */
--text-primary: #1D1D1F (主文字)
--text-secondary: #86868B (次要文字)
--bg-primary: #FFFFFF (主背景)
--bg-secondary: #F2F2F7 (次背景)
--bg-grouped: #F2F2F7 (分组背景)

/* 暗色模式 */
--text-primary: #F2F2F7
--bg-primary: #000000
--bg-secondary: #1C1C1E
```

- 字体：SF Pro Text/Display (iOS 系统字体)
- 标题：H1(28px) H2(24px) H3(20px) H4(18px)
- 正文：Body(16px) Secondary(14px) Caption(12px)
- 字重：标题用 700，强调用 600，正文用 400

### 间距系统
- 基础间距：4px 8px 12px 16px 20px 24px 32px
- 页面边距：16px
- 卡片内边距：16px
- 组件间距：12px

### 圆角规范
- 按钮：12px
- 卡片：16px
- 输入框：12px
- 页面容器：38px (iPhone 屏幕圆角)

### 阴影系统
- 卡片阴影：0 4px 16px rgba(0,0,0,0.04), 0 2px 4px rgba(0,0,0,0.02)
- 悬浮阴影：0 8px 32px rgba(0,0,0,0.08), 0 4px 8px rgba(0,0,0,0.04)

## 📱 页面清单与核心功能

### 1. 专注首页 `/(tabs)/index`
**核心功能**：展示当前专注状态，提供快速开始入口
**设计要点**：
- 顶部状态 Banner（进行中显示倒计时，暂停显示恢复时间）
- 快速开始卡片（主 CTA，推荐时长提示）
- 目标应用预览（TokenLabel 图标列表，显示数量）
- 功能入口区（任务面板、应用管理、权限设置）
- 空态处理（未选目标/未创建计划的引导）

**状态变化**：
- 未开始：显示"开始专注"按钮
- 进行中：Banner 显示蓝色渐变 + 倒计时
- 暂停中：Banner 显示橙色 + "X分钟后恢复"

### 2. 快速开始（模态） `/quick-start/index`
**核心功能**：最少步骤完成一次性专注启动
**设计要点**：
- 毛玻璃模态背景
- 时长选择：预设按钮(5/15/25/45/90) + 自定义滑杆
- 目标预览：已选应用图标 + 管理入口
- 底部主按钮："开始专注/继续专注"
- 次要按钮："去任务面板"

**交互状态**：
- 未授权：按钮置灰 + 提示文案 + "去权限设置"
- 未选目标：按钮置灰 + "请先选择专注目标"
- 正常：主按钮可点击，显示渐变效果

### 3. 任务面板 `/plans/index`
**核心功能**：管理周期性专注计划
**设计要点**：
- 顶部统计信息（计划数量、状态说明）
- 计划卡片列表：
  - 启用状态：绿色背景 + 成功图标
  - 停用状态：红色背景 + 暂停图标
- 24小时时间线可视化：
  - 整点虚线网格
  - 当前时间高亮线
  - 专注/屏蔽时段色条
- 空态：引导创建第一个计划

**时间线设计**：
- 高度：1200px (50px/小时)
- 专注模式：绿色渐变条
- 屏蔽模式：红色渐变条
- 当前时间：蓝色发光线

### 4. 新建/编辑计划 `/plans/add`
**核心功能**：创建或编辑周期性计划
**设计要点**：
- 表单字段：
  - 计划名称（可选输入框）
  - 重复规则（周几选择器，圆形按钮组）
  - 开始/结束时间（时间选择器）
  - 目标预览（应用图标 + 管理入口）
- 底部保存按钮（主色渐变）
- 表单验证提示

**周几选择器**：
- 7个圆形按钮横向排列
- 选中：主色背景 + 白字
- 未选中：灰色边框 + 灰字

### 5. 权限设置 `/setting/permission`
**核心功能**：引导用户开启必需权限
**设计要点**：
- 权限卡片列表：
  - 屏幕使用时间（必需）：红色警告样式
  - 通知权限（建议）：蓝色信息样式
- 每个卡片包含：
  - 权限名称 + 图标
  - 状态说明（已开启/未开启）
  - "去设置"按钮
- 权限说明文案

### 6. 我的页面 `/(tabs)/user`
**核心功能**：用户信息与设置入口
**设计要点**：
- 用户信息卡：头像 + 昵称 + 编辑入口
- 功能入口列表
  - 会员中心
  - 金币中心
  - 设置
- 统一的列表项样式（图标 + 标题 + 箭头）

### 7. 登录系列页面
#### 7.1 登录起始 `/login/start`
**核心功能**：展示价值点，引导登录
**设计要点**：
- 价值主张卡片
- 登录方式按钮组（登录/注册/微信）

#### 7.2 登录页 `/login/index`
**核心功能**：账号密码登录
**设计要点**：
- 简洁表单（手机号/邮箱 + 密码）
- 主按钮"登录"
- 辅助链接（注册/忘记密码）

#### 7.3 注册页 `/login/register`
**核心功能**：新用户注册
**设计要点**：
- 最小字段集（避免流失）
- 密码确认
- 协议同意

#### 7.4 微信登录 `/login/wx`
**核心功能**：第三方登录流程
**设计要点**：
- 授权状态提示
- 微信品牌色
- 失败重试机制

### 8. 拦截页面 (ShieldExtension)
**核心功能**：被屏蔽应用的拦截界面
**设计要点**：
- 深色背景（减少吸引力）
- 核心信息："已被屏蔽"
- 时间信息："总时长 X 分钟，HH:mm 结束"
- 简单返回按钮

### 9. 异常页面 `+not-found`
**核心功能**：404错误处理
**设计要点**：
- 友好的错误提示
- 返回首页按钮

## 📐 设备规范

### iPhone 15 Pro 规范
- 屏幕尺寸：390×844 points
- 安全区域：顶部 44px，底部 34px
- 状态栏高度：44px
- 导航栏高度：56px
- Dynamic Island：126×30px，圆角 16px

### 响应式考虑
- 最小点击区域：44×44px
- 文字最小尺寸：12px
- 对比度：≥4.5:1
- 支持动态字体缩放

## 🎯 交互细节

### 微动画
- 页面进入：fadeInUp 0.4s ease-out
- 按钮点击：scale(0.98) 0.1s
- 卡片悬停：translateY(-2px) + 阴影加深
- Toast 动画：滑入滑出 + 透明度变化

### 反馈机制
- 成功操作：绿色 Toast + 成功图标
- 错误提示：红色 Toast + 错误图标
- 加载状态：按钮内 loading 动画
- 空态引导：插画 + 引导文案 + CTA

### 手势支持
- 模态下拉关闭
- 列表项左滑操作
- 长按删除确认

## 🔧 技术实现建议

### CSS 变量系统
使用 CSS Custom Properties 实现主题切换和设计 token 管理

### 组件复用
- 统一的按钮组件（支持多种状态）
- 可配置的卡片组件
- 通用的表单字段组件

### 性能优化
- 图标使用 SVG 或 Icon Font
- 渐变使用 CSS 而非图片
- 动画使用 transform 和 opacity

## 📝 文案规范

### 按钮文案
- 主操作：动词 + 对象（"开始专注"、"保存计划"）
- 次操作：简洁明确（"取消"、"稍后"）
- 危险操作：明确后果（"删除计划"、"清空数据"）

### 状态文案
- 进行中："🎯 专注进行中 · 剩余 XX:XX"
- 暂停中："⏸️ 已暂停 · XX:XX 后恢复"
- 已结束："✅ 专注完成 · 坚持了 XX 分钟"

### 空态文案
- 未选目标："选择专注目标，开始你的专注之旅"
- 无计划："创建你的第一个周期计划"
- 权限未授权："开启权限，体验完整功能"

---

**使用说明**：基于此提示词生成设计稿时，请严格遵循色彩、字体、间距、圆角等设计规范，确保页面间视觉一致性。每个页面都应体现对应的核心功能，并考虑各种交互状态的视觉反馈。

## 新版设计提示词

设计一个现代感十足的移动应用界面，具有以下特点：

1. 采用深色主题设计，背景色为深灰色或纯黑色
2. 顶部使用红橙色到青绿色的渐变背景，营造视觉冲击力
3. 所有UI元素采用圆角矩形设计，圆角半径适中
4. 卡片式布局，每个功能模块独立成卡片，带有半透明毛玻璃效果
5. 使用高对比度配色，白色文字配深色背景
6. 主要操作按钮采用橙红色到绿色的渐变效果
7. 图标与文字结合，提升用户体验
8. 垂直排列布局，适合移动端单手操作
9. 合理的留白和间距，避免界面拥挤
10. 状态切换元素（如开关）有明确的视觉反馈
11. 字体层级清晰，重要信息突出显示
12. 整体风格简约现代，符合iOS设计规范
