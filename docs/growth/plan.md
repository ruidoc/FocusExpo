# 增长策略

> 说明：以下方案基于“专注/自律/时间管理（含 iOS 屏幕使用时间能力）”类应用的通行业务经验与当前仓库代码线索（存在屏幕时间相关扩展与微信登录）。部分平台资质与投放可用性因地区而异，标注“需要验证”的地方上线前需确认。

### 目标与阶段

- **阶段目标（0-4 周）**：快速获取首批种子用户 3,000-5,000，验证核心留存与付费路径
- **核心北极星指标**：第 1 次专注会话完成率、D1 留存、7 日内试用开启率、试用转会员率
- **渠道配比（首月建议）**：
  - 付费投放 60%-70%（快速拿量，校准人群与创意）
  - 自然增长 30%-40%（ASO、内容、社群、推荐）

### 关键用户画像与价值主张

- **画像 A ｜学生/备考**：高中/大学/考研人群，需求：提升专注、刻意练习、减少刷短视频
  - 信息框架：提升成绩/备考效率 → 有组织的专注计划 → 可量化打卡与复盘
  - 证据：用户故事、学习时长榜、连续打卡徽章
- **画像 B ｜职场/独立开发者**：希望提高深度工作时间与产出
  - 信息框架：减少上下文切换 → 阻断干扰 → 周报与产出复盘
- **画像 C ｜家长**（若产品支持监护）：希望孩子自律用机
  - 信息框架：时间配额/白名单 → 家校协同 → 周报与奖励

### 渠道组合总览

- **付费投放（建议优先）**：

  - 腾讯广告（微信朋友圈、公众号、优量汇）：CPI 预估 ¥15-35（需要验证；受地域和素材影响）
  - 巨量引擎（抖音/今日头条）：CPI 预估 ¥10-30（需要验证）
  - 小红书付费合作/KOL 笔记加热：CPE/CPV 转化至安装 CPI 约 ¥20-50（需要验证）
  - Apple Search Ads：如面向非中国大陆市场，CPI 约 $0.8-$2.5（需要验证地区投放可用性）

- **自然增长**：
  - ASO：中文关键词（“专注、时间管理、番茄钟、自律、防沉迷”）与英文关键词（“focus, productivity, pomodoro, screen time”）
  - 内容矩阵：小红书/B 站/抖音图文短视频，主题“专注挑战/自律复盘/学习视角 vlog”
  - 社群与私域：微信/QQ/飞书群每日打卡，周赛季赛榜单
  - 产品内增长：战报海报、周报卡片、徽章里程碑分享、邀请返利

### 付费投放策略（首月）

1. 账户与归因

- 归因与分析：AppsFlyer/Adjust 或 Apple 自带+Firebase（需要验证当前埋点方案）
- 事件：install → 注册 → 首次专注开始/完成 → 开启试用 → 订阅付费 → N 日留存

2. 人群与素材

- 人群切分：学生/备考、职场深度工作、家长（若支持）
- 素材 3-5 套/人群：
  - 痛点切入：刷短视频 → 效率低 → 用 X 天提升专注
  - 功能展示：一键专注、应用拦截、白名单、周报
  - 社会证据：连续打卡、用户好评、学习时长排行
- 落地页/商详：商店截图 AB 测（痛点型 vs 成就型）、首屏价值+社会证据+流程 3 步

3. 出价与节奏

- 周预算建议：¥20,000-40,000（按 CPI ¥15-30 预估 650-2,600 installs/周）
- 第 1 周：多素材小预算探索（10+ 素材包，单包日预算 ¥300-800），快速淘汰
- 第 2-3 周：放大 2-3 个优胜素材/人群；并行测试落地页/商详
- 第 4 周：稳定投放+引入 LAL（类人群）与重定向（浏览商详未安装/注册未首专）

4. 关键 KPI（首月建议值）

- CPI ≤ ¥25（人群/素材维度看板）
- 注册率 ≥ 65%，首次专注完成率 ≥ 60%
- 7 日试用开启率 ≥ 8%-12%，试用转订阅率 ≥ 25%-35%

### 自然增长策略

1. ASO/商店优化

- 标题/副标题：突出“专注/时间管理/番茄钟/防沉迷”
- 截图 AB：功能页 vs 成就海报；添加中文/英文双语（面向多市场时）
- 评分与评语：产品内弹窗在“完成第 3 次专注后”触发请求评分

2. 内容与 KOL/KOC

- 小红书：主题模板（“30 天专注挑战”“考研自律的一天”“专注周报展示”）
- B 站：长图文/视频“如何提升深度工作时间”“防止刷短视频干扰”
- 抖音：15-30 秒工具型演示+对比（开关白名单前后效率）
- 节奏：每周至少 6-8 篇笔记/视频；1-2 个腰部 KOL 合作（投后沉淀进社群）

3. 社群与裂变

- 微信/QQ/飞书群：每日番茄打卡、周赛季赛、学习搭子互助
- 邀请奖励：
  - 被邀请人注册+首次专注完成 → 邀请人获 VIP 7 天
  - 首次付费 → 双方各加赠 1 个月（上限 3 个月）
- 海报分享：生成“周报/成就海报”一键保存/分享（适配 9:16/1:1）

4. 合作渠道

- 高校社团/图书馆/网课机构联名：兑换码/专属榜单/联合挑战赛
- 生产力社区（少数派、V2EX 等）：评测与体验兑换

### 种子用户计划（0-4 周）

- **目标**：获取 3,000-5,000 安装，沉淀 1,000-2,000 活跃种子用户
- **激励**：VIP 1-3 个月、徽章/排名、纪念周边（预算内小额）
- **招募渠道**：高校社群、学习类 KOL 评论区、小红书/B 站话题、效率社区
- **转化路径**：招募页 → 建群/入群问卷 → 安装 → 新手任务（完成 3 次专注）→ 发放 VIP
- **共创机制**：每周线上碰头/问卷，Roadmap 透明；提交有效建议可获额外时长
- **复盘**：每周 D1/D7 留存、首专转化、NPS；产出公开周报

时间线：

- 周 1：发布招募贴（2-3 平台），首批 300-500 人入群
- 周 2：新手任务与奖励上线，集中答疑/产品说明会
- 周 3：挑战赛启动（7 天），颁发徽章+海报扩散
- 周 4：精选用户故事，对外内容投放与商详优化

### 预算与 ROI 预估（首月）

> 说明：以下为敏感度区间估算，真实取值受创意、人群、商详与产品表现影响较大，需以归因数据校准。

- 总预算：¥50,000 - ¥80,000
- 付费投放：¥40,000 - ¥60,000（CPI ¥15-30 → 1,300 - 4,000 installs）
- KOL/内容：¥8,000 - ¥15,000（含素材制作/笔记加热）
- 激励成本：¥2,000 - ¥5,000（周边/工具/直播活动）

转化与收入（示例“基线”场景）：

- 注册率 65%，首专完成率 60%
- 7 日试用开启率 10%，试用转订阅率 30%
- 首月订阅价格（示例）：¥25-38（需要验证产品定价与分成）
- 预计首月新增订阅数： installs × 0.65 × 0.60 × 0.10 × 0.30
- 若 installs=3,000，则新增订阅 ≈ 35；首月订阅收入 ≈ ¥875-¥1,330
- 随留存与续费提升，3 个月 LTV 可达首月的 2-4 倍（需要验证留存与续费曲线）

注：首月 ROI 可能不闭环，目标是找到“可放大的正向组合”（创意 × 人群 × 商详 × 留存），第 2-3 个月在留存与复购改善后追求回本。

## AI 驱动的增长方案（自动化与智能化落地）

> 说明：以下方案以“尽可能用 AI 自动化”作为原则，覆盖投放、ASO、实验、生命周期触达、社群内容、客服与风控。所有成本与可行性“需要验证”，上线前应小范围试点与 A/B 校准。

### 1. 广告创意与落地页：AI 生成-筛选-迭代闭环

- 目标：把素材产能与测试效率提升 3-5 倍，降低 CPI 10%-30%
- 工具与链路：
  - 文案/脚本：大语言模型（如 GPT 家族/通义千问/DeepSeek）+ 模板库
  - 画面与视频：图像生成/编辑（Midjourney/Stable Diffusion）+ 短视频生成（CapCut 脚本自动化、Runway、Pika）
  - 自动配音/字幕：科大讯飞/ElevenLabs + SRT 生成
  - 动态落地页：低代码页面（如 Notion/Framer/Locofy）+ 模型生成文案/FAQ
  - 评估：自研或 BI 看板接归因数据（点击 → 安装 → 注册 → 首专 → 试用 → 订阅）
- 流程：
  1. 画像 × 价值点 → 生成 10-20 条短脚本与 KV
  2. 自动合成 10+ 短视频/海报；一键出多尺寸/多语种
  3. 小预算并行探测；每 48 小时淘汰底部 30%
  4. 让模型基于数据总结“胜出要素”，自动生成下一轮变体
- 成本预估（月）：
  - 模型与工具订阅：¥1,000-3,000（依并发与团队规模）
  - 人力：1 运营+1 设计可降至 0.5 人天/周维护（需要验证）

### 2. ASO 自动化：关键词与商详智能优化

- 关键词发现：抓取评论/竞品文案 → 模型提取长尾词 → 频次与相关性评分
- 标题/副标题/关键词字段：模型生成多个组合，按地区与画像差异化
- 截图与描述：模型根据“本周胜出要素”自动改写描述，输出截图文案层
- 评价回复：模型按情绪与问题类型自动生成回复草稿（人工审核后发布）
- 节奏：每周 1 次小幅更新，重大版本前做多方案 AB
- 成本：工具订阅 ¥300-800/月 + 0.5 人天/周

### 3. AI 驱动的实验与定价：多臂老虎机（Bandit）

- 应用：引导页文案/顺序、评分弹窗触发点、试用天数/价格、Paywall 结构
- 机制：基于多臂老虎机算法动态分配流量给表现更优的变体，缩短收敛时间
- 数据：实时计算注册 → 首专 → 试用 → 付费的转化率与置信区间，自动停更差变体
- 工具：自研轻量规则引擎 + 统计库（或使用现成 SDK，需评估集成复杂度）
- 预期：试用与付费路径转化 10%-25% 改善（区间视流量与噪声，需验证）

### 4. 生命周期智能触达：流失预测与消息编排

- 预测：基于“近 3 天未首专/近 7 天未完成专注/权限被关闭/学习时长下降”等特征做二分类流失预测（每天批量离线）
- 触达：
  - App 内：个性化提醒卡片/弹窗（合规与频控）
  - 推送：iOS/Android 通知定时与内容个性化（遵守系统限制）
  - 邮件/短信（若采集）：重激活/回流优惠（需要验证隐私与授权）
- 内容生成：模型基于“用户画像 × 最近行为 × 目标动作”生成个性化文案
- 编排：按周设定“新手 7 天旅程”“回流 14 天旅程”“会员续费 30 天旅程”
- 预期：D7 留存 +1-3pt，试用率 +10-20%（需要验证）

### 5. AI Onboarding 与客服：智能助手与知识库

- Onboarding：在首日引导中嵌入“AI 学习/专注教练”，用对话式引导完成 3 次首专
- 知识库：把 FAQ/隐私/权限/常见问题统一到向量知识库，模型检索生成答案
- 工单分流：模型对反馈进行意图分类与情感分析，优先级路由到人工
- 训练数据：来自用户问题、差评主题、社群高频问题
- 成本：向量数据库 + 模型调用 ¥500-2,000/月（按量）

### 6. 社群与内容自动化：日历、脚本与生成

- 内容日历：每周根据“胜出要素”自动产出下周小红书/B 站/抖音选题与脚本
- 海报/周报卡片：模型生成标题与亮点文案，自动拼版为 9:16/1:1 素材
- 社群主持：群内“AI 主持人”发布每日打卡主题、周赛提醒、回答基础问题
- KOC 协作：模型生成合作 Brief 与规范，自动发放任务与收稿质检清单

### 7. 风控与 UGC 审核：

- 防刷与作弊：对“邀请返利”监测设备指纹/IP 段/行为序列，模型做异常打分
- UGC 审核：对于公开榜单昵称/评论/笔记，模型做敏感词与相似度检测
- 合规：输出人工复核清单与争议样本回溯

### 8. 分析与数据工程（最小可用栈）

- 埋点：Firebase/Amplitude 事件，服务端打点对齐订阅与留存
- 看板：自动日报（获取/激活/付费/复购），模型生成高亮“异常与建议”
- 归因：AppsFlyer/Adjust 与商店数据对表；导出供模型训练

### 9. 成本与人力（首月粗估，需按地区与量级验证）

- 模型与工具订阅：¥2,000-6,000/月（图文/视频/向量库/编排/AB）
- 人力配置：
  - 增长负责人 0.5-1 人，运营/内容 1 人，设计 0.5 人，数据 0.5 人
  - 通过自动化可把素材产能与报表产能提升至 3-5 倍

### 10. 集成位置（结合当前项目结构）

- App 内：
  - `app/login/start.tsx` 与 `app/login/index.tsx`：引导文案 AB、评分弹窗时机
  - `components/home/focus-notice.tsx`：个性化提醒卡片
  - `components/native/TokenLabel.tsx`：成就/徽章展示的分享卡片文案注入
  - `utils/permission.ts`：权限引导提示语与时机实验
- 服务器侧（建议）：
  - 规则/编排服务：下发实验配置、旅程节点与消息文案
  - 事件回收：行为与订阅事件回传，供模型训练与看板

### 11. 验证步骤

- 周 1：用历史与竞品数据冷启动关键词/脚本；上线 10+ 素材小预算测试
- 周 2：接入旅程与个性化文案；开启 Paywall/引导 AB 或 bandit
- 周 3：落地社群 AI 主持人与周报卡片；上线风控基础规则
- 周 4：复盘：CPI、首专、试用、付费、留存；生成“胜出要素报告”与下一月计划
